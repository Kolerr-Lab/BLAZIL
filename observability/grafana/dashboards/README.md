# Grafana Dashboards

This directory contains Grafana dashboard definitions for Blazil monitoring.

## Dashboards

- `transaction-throughput.json` - Transaction processing metrics
- `service-health.json` - Service health and availability
- `latency-overview.json` - Latency percentiles and distribution
- `resource-utilization.json` - CPU, memory, network usage

## Provisioning

Dashboards in this directory are automatically provisioned when using the Docker Compose stack.
